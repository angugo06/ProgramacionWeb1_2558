Formas de agregar estilos a una pagina web

Estilos en linea

Estos estilos se agregan directamente en el elemento HTML mediante el atributo style

Este parrafo tiene un fondo agregado con estilos en linea

Estilos Internos

Estos eleentos se aregan en el mismo archivo elemento

Este elemento de be de estar dentro de la atiqueta <head>

Estilos Externos

Estos eleentos se aregan en diferente archivo; ofrecen modularidad, mantenimiento, orden, eficiencia y entre otras ventajas para tener una hoja de estilos externa dentro de la etiqueta <Se una etiqueta link>

Este elemento de be de estar dentro de la atiqueta <head>
