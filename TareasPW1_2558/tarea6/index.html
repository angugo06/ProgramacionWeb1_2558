HTML y sus contenedores semanticos

Con el siguiente material se espera lograr a un entendimiento de nivel basico de HTML y de los contenedores semantcios que hay, tales como div, header, section y footer.

Ideas principales de HTML

HTML

Lenguaje de marcado de hipertexto que se utiliza para la estructuracion de paginas web

Elemento HTML

Unidad basica de estructura y contenido de una pagina web, compuesta de:

  1. Etiqueta de apertura
  2. Pares atributo-valor
  3. contenido
  4. Etiqueta de cierre

Etiqueta <head>

Etiqueta para comunicarse on el navegador a traves de los metadatos, mejora el SEO y se enlazan archivos externos de estilos o scripts

Etiqueta <body>

Toda la informacion que el usuario va a visualizar

Contenedores semanticos

<header> Contenido introductorio de una pagina (a menudo contiene un logotipo, menu de navegacion u otra informacion de identificacion del sitio)
<nav> Representa una seccion de una pagina que contiene enlaces a otras paginas o secciones del sitio. Se usa para enlaces de navegacion que ayudan a los usuarios navegar por nuestro sitio web
<main> Representa el contenido principal de una pagina (no debe incluir ningun contenido que se repita en varias paginas como menus de navegacion, pies de pagina o barras laterales)
<section> Seccion de una pagina que agrupa contenido relacionado dentro del sitio web (se puede usar para dividir una pagina en varias seccioneso agrupar contenido dentro de una seccion mas grande)
<article> Unidad autonoma de contenido o historia que puede distribuirse o reutilizarse de forma independiente; por ejemplo, articulos de noticias o publicaciones de blogs
<footer> Contenido de cierre de una pagina o seccion/la parte inferior de una pagina o seccion (a menudo contine informacion de derechos de autor, contacto, ubicacion u otra informacion complementaria)

Elementos principales

Elemento <a>

Enlazar o navegar a otras paginas o secciones, hay tres tipos:

  • Internos, para navegar a enlaces nuestros
  • Externos, para navegar a enlaces externos
  • Anclaje, para navegar dentro de la misma pagina

Algunos valores especiales en su atributo HREF son:

href="https://wa.me/5519541638"
Whatsapp a un numero declarado
href="mailto:example@gmail.com"
Email a un correo declarado

Elemento <form>

Forma de comunicarse con la pagina web, usado para procesar datos de los usuarios y darles el uso apropiado
